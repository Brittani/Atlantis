概述

一路走来，项目组走过很多弯路，有过喜悦，有过失落，但是不变的坚持。在不断的探索、发现和总结中，我们持续的完善自己，完善团队的管理规范。
我们始终坚持规范以人为本，科学、敏捷；坚持管理以规范公正为基础，公平、效率；坚持一切为团队的根本利益服务，同舟共济。

  • Project Manage System

    PMS(高分项目管理系统)

    PMS是一个项目过程管理一体化平台。
    提供需求管理、软件设计、开发过程管理以及缺陷追踪等完整的项目生命周期管理支持。

    系统漫游

    PMS主要功能包含基于角色的权限控制、项目活动、新闻、讨论区、版本库、文档管理、以及便捷的文件分享。
    PMS是面向用户的设计，系统会将您所关注的信息汇集到用户工作台，并且会通过EMAIL向您发送最新的变更情报。您所需要做的就是Check邮件或者直接登陆到您的工作台处理和分配项目任务。
